# ludo
- When a new game is created, you can choose 2, 3, or 4 players. (system create automatically players and tokens)
- Players take turns one by one in order (Player 1 → Player 2 → ... → last player → back to Player 1).
- If a player rolls a 6, they get another turn.
- A token can only come out of home if the player rolls a 6.
- The board has 52 positions in a loop.
- Some positions are safe zones predefined : 1, 9, 14, 22, 27, 35, 40, 48
- If a token is on these spots, it cannot be attacked or sent home.
- Each player has 4 tokens to finish.
- The game ends
